# Project 4: Frontend and Backend Integration

## Overview

This project was developed as part of the DecodeLabs Full Stack Development Industrial Training Kit. The objective of this project is to integrate the frontend with backend APIs to build a complete full-stack application.

## Technologies Used

### Frontend

* HTML
* CSS
* JavaScript

### Backend

* Node.js
* Express.js

### Database

* MySQL

## Features

* User-friendly frontend interface
* Frontend to backend communication using Fetch API
* Asynchronous requests using async/await
* Dynamic UI updates
* JSON parsing and serialization
* Error handling using try/catch
* Database integration
* User management system

## Functionalities

### Add User

Users can enter their details through the frontend form and save them to the database.

### View Users

All users stored in the database are dynamically displayed on the webpage.

### Delete User

Users can be removed directly from the frontend interface.

## Project Workflow

Frontend (HTML/CSS/JavaScript)
↓
Fetch API Requests
↓
Express Backend
↓
MySQL Database
↓
JSON Response
↓
Dynamic DOM Updates

## Installation

1. Clone the repository.
2. Install dependencies:

npm install

3. Configure MySQL database.
4. Start the server:

node server.js

5. Open the application in the browser:

http://localhost:3000

## Output

The application provides complete frontend and backend integration where users can interact with the database through a web interface.
